Kykylauncher

Le launcher Minecraft pour rejoindre le serveur en un clic.

Télécharger Kykylauncher.exe
Windows · launcher.exe

Installation

  1. Télécharge le fichier ci-dessus
  2. Si Windows affiche un avertissement SmartScreen, clique sur "Informations complémentaires" puis "Exécuter quand même"
  3. Lance Kykylauncher.exe, entre ton pseudo et clique sur Jouer
